Acerca de Variables

Que son variables, Hoisting, Nombres, Scope

Tipo de variables

Las variables existentes son: let, var y const

Su declaracion

Para definir tipos de datos

let + nombre variable = valor;

let es una variable la cual es utilizada como variable local con un scope local

const + nombre variable = valor;

const es una constante con un scope local y su valor es invariable

var + nombre variable = valor;

var es una variable con un scope global y la cual se inicializa con un valor inicializacion

Hoisting

Paracticamente es que la variable o funcion cual queremos que arroje algun dato en concreto si su valor o inicializacion esta por delante a su iteracion esta sera undefined.

Nombre para Variables

EL definir las variables usualmente se aplican en minusculas y las constantes en mayusculas, alemenos en Python es asi pero en javascript existe la palabra const por lo que pude usarse el uso de minusculas en las constantes, una muy buena manera de definir variables es con dos tipos: CamelCase y guiones_bajos, ademas es importante que las variables en cualquier lenguaje de programacion sean claras y precisas ya que es mejor menos comentarios y mas variables claras.

Variable Scopes

Hacen referencia a la visibilidad de la varaible o como puede ser su uso despues de declararla existen tres tipos de Scope
Block, Function, Global

Block: Block es relacionado unicamente a que es scope se mantendra unicamente en el contexto que este, ejemplo: si esta dentro de un IF solo es contexto de ese IF

Function: Function es relacionado unicamente a que es scope se mantendra unicamente en el contexto de una funcion y unicamente existe ahi, ejemplo: si esta dentro de un function solo es contexto de ese function

Global: Global es un scope se puede usar en todo tipo de contexto ya sea en dentro o fuera de bloques aunque usar var hoy en dia es mala practica hacerlas dentro de un bloque y usarlas fuera.
