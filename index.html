Nombre para Variables

EL definir las variables usualmente se aplican en minusculas y las constantes en mayusculas, alemenos en Python es asi pero en javascript existe la palabra const por lo que pude usarse el uso de minusculas en las constantes, una muy buena manera de definir variables es con dos tipos: CamelCase y guiones_bajos, ademas es importante que las variables en cualquier lenguaje de programacion sean claras y precisas ya que es mejor menos comentarios y mas variables claras.

Variable Scopes

Hacen referencia a la visibilidad de la varaible o como puede ser su uso despues de declararla existen tres tipos de Scope
Block, Function, Global

Block: Block es relacionado unicamente a que es scope se mantendra unicamente en el contexto que este, ejemplo: si esta dentro de un IF solo es contexto de ese IF

Function: Function es relacionado unicamente a que es scope se mantendra unicamente en el contexto de una funcion y unicamente existe ahi, ejemplo: si esta dentro de un function solo es contexto de ese function

Global: Global es un scope se puede usar en todo tipo de contexto ya sea en dentro o fuera de bloques aunque usar var hoy en dia es mala practica hacerlas dentro de un bloque y usarlas fuera.

Tipos de datos en variables

Los tipos de datos en variables se puden categorizar en string, undefined, number, bigint, boolean, null

Estos tipos de datos son llamados "tipos primitivos" debido a que son datos y no objeto con metodos como tal.

Estos tipos de datos ademas pueden almacenarse en cualquiera de las 3 variables ya nombradas let, var, const

String: el string es una cadena de caracteres la cual en los lenguajes de programacion son para representar texto.

Undefined: es el tipo de dato el cual te rebotara cuando no es incializada la variable antes de su ejecucion esto puede pasar por Hoisting.

Number y BigInt: el tipo de dato number/BigInt es cuando queremos que no sean en este caso cadena de texto sino que se trabaje como number o mejor conocido como integer (entero) o float (decimal).

Boolean: el tipo boolean nos mas que nada un si / no en javascritp es conocido como true o false

Null: el null no es un tipo de dato como tal sino es mas bien nada es un dato arrojado por el lenguaje cuando en la variable se encunetra inicializada pero no continene ningun dato almacenado.

Bajo este mismo concepto se puede decir que Javascript esta construido por un JSON usando clave y valor en valor pueden haber mas claves y valor con sus respectivos metodos. Por lo que todos las funciones Built In en javascript son hijas del padre llamado Window.

Operadores de comparasion

Los comparadores de valor son 3 los cuales son ==, ===, Object.is

El operador ==: compara que el valor sea el mismo valor.

El operador ===: compara que el valor sea el mismo valor y ademas el mismo tipo de dato.

El operador Object.is(): compara que el valor sea el mismo valor y ademas el mismo tipo de dato pero este convalida los valores incaluclables
